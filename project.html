About the Akan

This is a website that helps users to know their Akan names. This is based on the user being able to input their birthdays and their gender.The gender helps the website determine the Akan name for the user.

Male Names

  1. Sunday: Kwasi
  2. Monday: Kwadwo
  3. Tuesday: Kwabena
  4. Wednesday: Kwaku
  5. Thursday: Yaw
  6. Friday: Kofi
  7. Saturday: Kwame

Female Names

  1. Sunday: Akosua
  2. Monday: Adwoa
  3. Tuesday: Abenaa
  4. Wednesday: Akua
  5. Thursday: Yaa
  6. Friday: Afua
  7. Saturday: Ama
